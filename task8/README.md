ГРАФ - ОРИЕНТИРОВАННЫЙ ГРАФ
================================

ОПИСАНИЕ
--------
Реализация ориентированного графа на языке C++ с использованием матрицы смежности.
Граф поддерживает добавление и удаление вершин и ребер, проверку достижимости,
поиск сильно связных компонент, а также операции объединения и разности графов.

СТРУКТУРА ПРОЕКТА
-----------------
graph.h      - объявление класса graph
graph.cpp    - реализация методов класса
main.cpp     - пользовательский интерфейс и тесты
Makefile     - файл для сборки

КОМПИЛЯЦИЯ И ЗАПУСК
-------------------
make         - компиляция и запуск тестов
make test    - компиляция и запуск тестов
make clean   - очистка скомпилированных файлов
./graph      - интерактивный режим
./graph test - режим тестирования

КЛАСС GRAPH
-----------
Конструкторы:
  graph()                           - создает пустой граф
  graph(const graph& other)         - конструктор копирования

Деструктор:
  ~graph()                          - освобождает память

Операторы:
  operator=(const graph& other)     - присваивание
  operator+(const graph& other)     - объединение графов
  operator+(int vertex)             - добавление вершины
  operator+(const int edge[2])      - добавление ребра
  operator-(const graph& other)     - разность графов
  operator-(int vertex)             - удаление вершины
  operator-(const int edge[2])      - удаление ребра
  operator<<                         - вывод графа

Методы:
  add_vertex(int v)                 - добавить вершину
  add_edge(int from, int to)        - добавить ребро
  add_both(int v1, int v2)          - добавить двунаправленное ребро
  remove_vertex(int v)              - удалить вершину
  remove_edge(int from, int to)     - удалить ребро
  remove_both(int v1, int v2)       - удалить двунаправленное ребро
  print_edges(int v)                - вывести исходящие ребра
  reachable(int from, int to)       - проверить достижимость
  find_scc()                        - найти сильно связные компоненты
  has_vertex(int v)                 - проверить наличие вершины
  has_edge(int from, int to)        - проверить наличие ребра
  get_size()                        - получить количество вершин
  get_vertex(int index)             - получить вершину по индексу

ОБРАБОТКА ОШИБОК
----------------
Все методы могут выбрасывать исключение graph::error с указанием строки ошибки.

ТЕСТЫ
-----
1. Базовые операции - добавление/удаление вершин и ребер
2. Операторы + и - - объединение и разность графов
3. Поиск сильно связных компонент - алгоритм Косарайю
