Shell

Командный интерпретатор (shell), поддерживающий конвейеры, перенаправление ввода/вывода и фоновое выполнение процессов.

Содежание:

    Makefile
        Сборка
    main.c             
        Основная логика shell
    token_list.c       
        Работа с лексикой
    token_list.h       
        Заголовочный файл для token_list.c

Makefile
    
    Переменные:

    CC = gcc - компилятор C
    CFLAGS = -Wall - флаги компиляции
    SOURCES = main.c token_list.c - исходные файлы
    OBJECTS = $(SOURCES:.c=.o) - объектные файлы
    EXECUTABLE = shell - имя исполняемого файла

    Цели:

    all - основная цель сборки
    $(EXECUTABLE) - компиляция исполняемого файла
    clean - очистка собранных файлов

main.c

    execute_conv() - выполняет конвейер команд с pipe и перенаправлениями
    execute_cmd() - выполняет команду (встроенный cd или внешнюю)
    parse_cmd() - разбирает токены на команды, файлы и флаги
    conv_append() - добавляет команду в конвейер
    welcome_user() - показывает приглашение с текущей директорией
    kill_zombie() - обработчик SIGCHLD для зомби-процессов

token_list.c

    read_tokens() - лексический анализ: читает и разбивает ввод на токены
    getsym() - читает символы с буферизацией
    add_to_list() - добавляет токен в динамический массив
    is_input_finished() - проверяет конец ввода
    sort_list() - сортирует токены
    delete_list() - освобождает память

token_list.h

    Содержит объявления структур и функций для работы с токенами.
