README — Итерируемые объекты на C++
Описание проекта

Учебная программа, демонстрирующая:

    Абстрактный базовый класс iterable_object с чисто виртуальными методами length() и contains().

    Класс string (строка, поиск подстроки).

    Шаблонный класс list<T> (динамический массив с методами add, get, contains).

    Класс string_list, унаследованный от string и list<string> (множественное виртуальное наследование).

    Полиморфное хранение объектов в std::vector<iterable_object*>.

    Интерактивное меню для создания, проверки и удаления объектов.

Программа позволяет создать несколько объектов разных типов, затем по одному поисковому запросу проверить каждый из них на содержание заданной строки.
