Список программ

1. Расширенный список файлов (аналог ls)

Используемые функции:

    get_mode_str() - преобразование прав доступа в строковый формат

    print_cur_path() - вывод текущего пути

    list_dir() - рекурсивный вывод содержимого директории

    parse_options() - обработка параметров командной строки

Программа:

    Отображение файлов и директорий

    Рекурсивный обход (-R)

    Детальная информация (-l) - права доступа, владелец, размер

    Информация о группе (-g)

    Поддержка специальных файлов

2. Вывод аргументов (аналог echo)

Программа:

    Вывод всех переданных аргументов

    Автоматическое добавление пробелов между аргументами

    Завершение вывода переводом строки

3. Текущая директория (аналог pwd)

Программа:

    Вывод абсолютного пути текущей рабочей директории

    Обработка ошибок получения пути

    Корректное освобождение памяти

4. Копирование файлов (аналог cp)

Программа:

    Бинарное копирование файлов

    Создание файла назначения при необходимости

    Буферизированное чтение/запись

    Обработка ошибок открытия файлов

5. Сравнение файлов (аналог cmp)

Программа:

    Побайтовое сравнение двух файлов

    Определение первой позиции различия

    Сообщение о идентичности файлов

    Обработка файлов разной длины

6. Просмотр файлов (аналог cat)

Используемые функции:

    parse_options() - обработка параметров командной строки

Программа:

    Вывод содержимого одного или нескольких файлов

    Нумерация строк (-n)

    Последовательная обработка нескольких файлов

    Продолжение работы при ошибках открытия файлов
